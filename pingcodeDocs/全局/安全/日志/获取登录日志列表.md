# 获取登录日志列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/security/login_logs`

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| logged_between | String | 是 | 登录时间介于的时间范围，通过','分割起始时间。 |
| user_ids | String | 否 | 成员id的列表，使用','分割，最多只能20个。 |
