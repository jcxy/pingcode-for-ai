# 获取团队中的成员列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/directory/groups/{group_id}/members`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| group_id | String | 是 | 团队id。 |
