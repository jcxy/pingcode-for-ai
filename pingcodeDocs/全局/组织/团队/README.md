# 团队

## API 列表

- [创建一个团队](创建一个团队.md) - `POST /v1/directory/groups`
- [部分更新一个团队](部分更新一个团队.md) - `PATCH /v1/directory/groups/{group_id}`
- [获取团队列表](获取团队列表.md) - `GET /v1/directory/groups`
- [向团队中添加一个成员](向团队中添加一个成员.md) - `POST /v1/directory/groups/{group_id}/members`
- [获取团队中的成员列表](获取团队中的成员列表.md) - `GET /v1/directory/groups/{group_id}/members`
- [在团队中移除一个成员](在团队中移除一个成员.md) - `DELETE /v1/directory/groups/{group_id}/members/{member_id}`
