# 上传一个代码段

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/attachments`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 请求参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| principal_type | String | 是 | 附件主体的类型。 |
| principal_id | String | 是 | 附件主体的id。 |
| comment_id | String | 否 | 附件主体的评论id。当需要向附件主体的评论上传文件或者代码段时，需要传入该参数。 |
| title | String | 是 | 代码段的标题。 |
| format | String | 是 | 代码段的语言。 |
| content | String | 是 | 代码段的内容。 |
