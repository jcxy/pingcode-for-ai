## API 列表

- [获取附件列表](获取附件列表.md) - `GET /v1/attachments?principal_type={principal_type}&principal_id={principal_id}`
- [删除一个附件](删除一个附件.md) - `DELETE /v1/attachments/{attachment_id}?principal_type={principal_type}&principal_id={principal_id}`
