# 删除一个关联

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/relations/{relation_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| relation_id | String | 是 | 关联的id。 |
