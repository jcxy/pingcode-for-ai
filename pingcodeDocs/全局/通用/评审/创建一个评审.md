# 创建一个评审

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/reviews`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| title | String | 是 | 评审的标题。 |
| pilot_id | String | 是 | 评审主体所在产品、项目或测试库的id。 |
| principal_type | String | 是 | 评审主体的类型。 |
| description | String | 否 | 评审的描述。 |
