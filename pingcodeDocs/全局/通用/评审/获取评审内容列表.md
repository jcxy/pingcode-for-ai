# 获取评审内容列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/reviews/{review_id}/principals?principal_type={principal_type}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| review_id | String | 是 | 评审的id。 |

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| principal_type | String | 是 | 评审主体的类型。 |
