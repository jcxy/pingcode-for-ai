# 获取活动记录列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/activities?principal_type={principal_type}&principal_id={principal_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| principal_type | String | 是 | 主体的类型。 |
| principal_id | String | 是 | 主体的id。 |
