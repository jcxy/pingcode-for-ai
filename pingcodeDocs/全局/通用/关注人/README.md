# 关注人

## API 列表

- [添加一个关注人](添加一个关注人.md) - `POST /v1/participants`
- [获取关注人列表](获取关注人列表.md) - `GET /v1/participants?principal_type={principal_type}&principal_id={principal_id}`
- [移除一个关注人](移除一个关注人.md) - `DELETE /v1/participants/{participant_id}?principal_type={principal_type}&principal_id={principal_id}`
