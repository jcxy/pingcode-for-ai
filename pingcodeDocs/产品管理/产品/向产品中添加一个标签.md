# 向产品中添加一个标签

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/ship/products/{product_id}/tags`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| product_id | String | 是 | 产品的id。 |

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 是 | 标签的名称。在一个产品中这个名称是唯一的。 |
