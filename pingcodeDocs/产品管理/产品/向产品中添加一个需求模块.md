# 向产品中添加一个需求模块

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/ship/products/{product_id}/suites`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| product_id | String | 是 | 产品的id。 |

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 是 | 模块名称。需求模块为树形结构，同一层次下的名称不能重复。 |
| type | String | 是 | 模块类型。允许值分别表示子产品和模块。 |
| parent_id | String | 否 | 父模块的id。 |
