# 在产品中移除一个需求模块

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/ship/products/{product_id}/suites/{suite_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 描述

请注意，删除一个模块会自动删除其所有的子模块。

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| product_id | String | 是 | 产品的id。 |
| suite_id | String | 是 | 模块id。 |
