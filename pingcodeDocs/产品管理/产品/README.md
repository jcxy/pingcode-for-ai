# 产品

## API 列表

- [创建一个产品](创建一个产品.md) - `POST /v1/ship/products`
- [部分更新一个产品](部分更新一个产品.md) - `PATCH /v1/ship/products/{product_id}`
- [获取产品列表](获取产品列表.md) - `GET /v1/ship/products`
- [向产品中添加一个成员](向产品中添加一个成员.md) - `POST /v1/ship/products/{product_id}/members`
- [获取产品中的成员列表](获取产品中的成员列表.md) - `GET /v1/ship/products/{product_id}/members`
- [在产品中移除一个成员](在产品中移除一个成员.md) - `DELETE /v1/ship/products/{product_id}/members/{member_id}`
- [向产品中添加一个需求模块](向产品中添加一个需求模块.md) - `POST /v1/ship/products/{product_id}/suites`
- [获取产品中的需求模块列表](获取产品中的需求模块列表.md) - `GET /v1/ship/products/{product_id}/suites`
- [在产品中移除一个需求模块](在产品中移除一个需求模块.md) - `DELETE /v1/ship/products/{product_id}/suites/{suite_id}`
- [获取产品中的需求排期列表](获取产品中的需求排期列表.md) - `GET /v1/ship/products/{product_id}/plans`
- [获取产品中的工单渠道列表](获取产品中的工单渠道列表.md) - `GET /v1/ship/products/{product_id}/channels`
- [获取产品中的工单类型列表](获取产品中的工单类型列表.md) - `GET /v1/ship/products/{product_id}/ticket_types`
- [获取产品中的标签列表](获取产品中的标签列表.md) - `GET /v1/ship/products/{product_id}/tags`
- [向产品中添加一个标签](向产品中添加一个标签.md) - `POST /v1/ship/products/{product_id}/tags`
- [在产品中移除一个标签](在产品中移除一个标签.md) - `DELETE /v1/ship/products/{product_id}/tags/{tag_id}`
