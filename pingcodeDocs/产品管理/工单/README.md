# 工单

## API 列表

- [创建一个工单](创建一个工单.md) - `POST /v1/ship/tickets`
- [部分更新一个工单](部分更新一个工单.md) - `PATCH /v1/ship/tickets/{ticket_id}`
- [获取工单列表](获取工单列表.md) - `GET /v1/ship/tickets`
- [获取工单类型列表](获取工单类型列表.md) - `GET /v1/ship/ticket/types?product_id={product_id}`
- [获取工单状态列表](获取工单状态列表.md) - `GET /v1/ship/ticket/states?product_id={product_id}`
- [获取工单属性列表](获取工单属性列表.md) - `GET /v1/ship/ticket/properties?product_id={product_id}`
- [获取工单渠道列表](获取工单渠道列表.md) - `GET /v1/ship/ticket/channels?product_id={product_id}`
- [获取工单优先级列表](获取工单优先级列表.md) - `GET /v1/ship/ticket/priorities?product_id={product_id}`
- [获取工单解决方案列表](获取工单解决方案列表.md) - `GET /v1/ship/ticket/solutions?product_id={product_id}`
- [获取工单标签列表](获取工单标签列表.md) - `GET /v1/ship/ticket/tags?product_id={product_id}`
- [获取工单流转记录列表](获取工单流转记录列表.md) - `GET /v1/ship/tickets/{ticket_id}/transition_histories`
