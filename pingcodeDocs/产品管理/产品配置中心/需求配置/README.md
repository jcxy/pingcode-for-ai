# 需求配置

## API 列表

- [获取全部需求状态列表](获取全部需求状态列表.md) - `GET /v1/ship/idea_states`
- [创建一个需求属性](创建一个需求属性.md) - `POST /v1/ship/idea_properties`
- [部分更新一个需求属性](部分更新一个需求属性.md) - `PATCH /v1/ship/idea_properties/{property_id}`
- [获取全部需求属性列表](获取全部需求属性列表.md) - `GET /v1/ship/idea_properties`
- [获取需求属性方案列表](获取需求属性方案列表.md) - `GET /v1/ship/idea_property_plans`
- [向需求属性方案中添加一个需求属性](向需求属性方案中添加一个需求属性.md) - `POST /v1/ship/idea_property_plans/{property_plan_id}/idea_properties`
- [获取需求属性方案中的需求属性列表](获取需求属性方案中的需求属性列表.md) - `GET /v1/ship/idea_property_plans/{property_plan_id}/idea_properties`
- [在需求属性方案中移除一个需求属性](在需求属性方案中移除一个需求属性.md) - `DELETE /v1/ship/idea_property_plans/{property_plan_id}/idea_properties/{property_id}`
