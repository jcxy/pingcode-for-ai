# 获取工单属性方案中的工单属性列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/ship/ticket_property_plans/{property_plan_id}/ticket_properties`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| property_plan_id | String | 是 | 工单属性方案的id。 |
