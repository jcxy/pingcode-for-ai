# 部分更新一个工单状态

## 基本信息

- **方法**: `Patch`
- **路径**: `/v1/ship/ticket_states/{ticket_state_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| ticket_state_id | String | 是 | 工单状态id。 |

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 否 | 工单状态的名称，在一个企业中这个名称是唯一的。 |
| type | String | 否 | 工单状态的类型。 |

## 成功响应

### 响应示例：

```json
{
  "id": "6422711c3f12e6c1e46d40f2",
  "url": "https://rest_api_root/v1/ship/ticket_states/6422711c3f12e6c1e46d40f2",
  "name": "已完成",
  "type": "completed",
  "color": "#56AB25"
}
```
