# 在状态方案中移除一个工单状态流转

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/ship/ticket_state_plans/{state_plan_id}/ticket_state_flows/{state_flow_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| state_plan_id | String | 是 | 工单状态方案的id。 |
| state_flow_id | String | 是 | 工单状态流转的id。 |
