# 在状态方案中移除一个工单状态

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/ship/ticket_state_plans/{state_plan_id}/ticket_states/{state_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 描述

移除状态后，每种类型的状态至少存在一种，否则将无法移除。

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| state_plan_id | String | 是 | 工单状态方案的id。 |
| state_id | String | 是 | 工单状态的id。 |
