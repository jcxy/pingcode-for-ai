# 部分更新一个客户

## 基本信息

- **方法**: `PATCH`
- **路径**: `/v1/ship/products/{product_id}/customers/{customer_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| product_id | String | 是 | 产品的id。 |
| customer_id | String | 是 | 客户的id。 |

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 否 | 客户的名称。 |
| assignee_id | String | 否 | 客户的负责人id。 |
| scale | Number | 否 | 客户的规模。 |
| description | String | 否 | 客户的描述。 |
