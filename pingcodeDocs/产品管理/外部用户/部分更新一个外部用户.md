# 部分更新一个外部用户

## 基本信息

- **方法**: `PATCH`
- **路径**: `/v1/ship/products/{product_id}/users/{user_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| product_id | String | 是 | 产品的id。 |
| user_id | String | 是 | 外部用户的id。 |

## 成功响应

### Success 200

| 字段 | 类型 | 描述 |
|------|------|------|
| customer_id | String | 外部用户所属客户的id。 |
