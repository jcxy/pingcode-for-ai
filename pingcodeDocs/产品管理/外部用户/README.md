# 外部用户

## API 列表

- [创建一个外部用户](创建一个外部用户.md) - `POST /v1/ship/products/{product_id}/users`
- [部分更新一个外部用户](部分更新一个外部用户.md) - `PATCH /v1/ship/products/{product_id}/users/{user_id}`
- [获取外部用户列表](获取外部用户列表.md) - `GET /v1/ship/products/{product_id}/users`
- [删除一个外部用户](删除一个外部用户.md) - `DELETE /v1/ship/products/{product_id}/users/{user_id}`
