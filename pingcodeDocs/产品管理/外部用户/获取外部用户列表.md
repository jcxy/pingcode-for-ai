# 获取外部用户列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/ship/products/{product_id}/users`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| product_id | String | 是 | 产品的id。 |
