# 获取需求模块列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/ship/idea/suites?product_id={product_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| product_id | String | 是 | 产品的id。 |

## 成功响应

{
  "page_size": 30,
  "page_index": 0,
  "total": 1,
  "values": [
    {
      "id": "63bb744414bd13c9def24ce4",

      "type": "module"
    }
