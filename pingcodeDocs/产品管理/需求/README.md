# 需求

## API 列表

- [创建一个需求](创建一个需求.md) - `POST /v1/ship/ideas`
- [部分更新一个需求](部分更新一个需求.md) - `PATCH /v1/ship/ideas/{idea_id}`
- [获取需求列表](获取需求列表.md) - `GET /v1/ship/ideas`
- [获取需求状态列表](获取需求状态列表.md) - `GET /v1/ship/idea/states?product_id={product_id}`
- [获取需求属性列表](获取需求属性列表.md) - `GET /v1/ship/idea/properties?product_id={product_id}`
- [获取需求模块列表](获取需求模块列表.md) - `GET /v1/ship/idea/suites?product_id={product_id}`
- [获取需求排期列表](获取需求排期列表.md) - `GET /v1/ship/idea/plans?product_id={product_id}`
- [获取需求优先级列表](获取需求优先级列表.md) - `GET /v1/ship/idea/priorities?product_id={product_id}`
- [获取需求流转记录列表](获取需求流转记录列表.md) - `GET /v1/ship/ideas/{idea_id}/transition_histories`
