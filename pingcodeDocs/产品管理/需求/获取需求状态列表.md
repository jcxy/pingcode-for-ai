# 获取需求状态列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/ship/idea/states?product_id={product_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| product_id | String | 是 | 产品的id。 |

## 成功响应

### 响应示例：

```json
{
  "page_size": 30,
  "page_index": 0,
  "total": 1,
  "values": [
    {
      "id": "63e1bf51898a0be5a2d21b2a",
      "url": "https://rest_api_root/v1/ship/idea_states/63e1bf51898a0be5a2d21b2a",
      "name": "待评审",
      "type": "pending"
    }
  ]
}
```
