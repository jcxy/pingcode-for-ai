# 向瀑布项目中添加一个工作项类型

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/project/projects/{project_id}/work_item_types`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 是 | 项目id。项目类型必须为`waterfall`，并且项目开启了本地配置。 |

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| id | String | 是 | 工作项类型的id，不支持增加计划类型（里程碑、阶段）。 |
