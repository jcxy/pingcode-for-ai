# 用例配置

## API 列表

- [获取全部用例类型列表](获取全部用例类型列表.md) - `GET /v1/testhub/case_types`
- [获取全部用例状态列表](获取全部用例状态列表.md) - `GET /v1/testhub/case_states`
- [获取全部用例属性列表](获取全部用例属性列表.md) - `GET /v1/testhub/case_properties`
- [获取全部重要程度列表](获取全部重要程度列表.md) - `GET /v1/testhub/case_important_levels`
- [获取用例属性方案列表](获取用例属性方案列表.md) - `GET /v1/testhub/case_property_plans`
- [获取属性方案中的用例属性列表](获取属性方案中的用例属性列表.md) - `GET /v1/testhub/case_property_plans/{property_plan_id}/case_properties`
- [创建一个用例属性](创建一个用例属性.md) - `POST /v1/testhub/case_properties`
- [部分更新一个用例属性](部分更新一个用例属性.md) - `PATCH /v1/testhub/case_properties/{property_id}`
- [向属性方案中添加一个用例属性](向属性方案中添加一个用例属性.md) - `POST /v1/testhub/case_property_plans/{property_plan_id}/case_properties`
- [在属性方案中移除一个用例属性](在属性方案中移除一个用例属性.md) - `DELETE /v1/testhub/case_property_plans/{property_plan_id}/case_properties/{property_id}`
