# 在属性方案中移除一个用例属性

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/testhub/case_property_plans/{property_plan_id}/case_properties/{property_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| property_plan_id | String | 是 | 测试用例属性方案的id。 |
| property_id | String | 是 | 测试用例属性的id。 |
