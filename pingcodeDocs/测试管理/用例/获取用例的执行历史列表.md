# 获取用例的执行历史列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/testhub/cases/{case_id}/histories`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 描述

获取测试用例所有执行用例的最后一次执行结果。

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| case_id | String | 是 | 测试用例的id。 |
