# 批量删除测试用例关联工作项

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/testhub/cases/{case_id}/work_item_relations/bulk`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| case_id | String | 是 | 测试用例id。 |

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| work_item_ids | String[] | 是 | 关联的工作项id数组。支持的工作项类型包括`bug`和`story`。 |
