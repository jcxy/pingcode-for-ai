# 获取测试库中的用例模块列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/testhub/libraries/{library_id}/suites`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| library_id | String | 是 | 测试库的id。 |

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| parent_id | String | 否 | 父测试模块的id。值可以是`root`或者某个模块id，当为空时，获取到所有的模块，当为`root`时，获取到所有的一级模块，当为某个模块id时，获取到该模块的直属子模块。 |
