# 获取测试库列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/testhub/libraries`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 否 | 测试库的名称。 |
| scope_type | String | 否 | 测试库的所属类型。 |
| scope_id | String | 否 | 测试库的所属id。 |
