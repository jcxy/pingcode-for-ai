# 向测试库中添加一个成员

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/testhub/libraries/{library_id}/members`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| library_id | String | 是 | 测试库的id。 |

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| member | Object | 否 | 测试库的成员。 |
| member.id | String | 是 | 企业成员或团队的id。 |
| member.type | String | 是 | 成员的类型。 |
| role_id | String | 否 | 角色的id。 |
