# 创建一个执行用例

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/testhub/runs`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| library_id | String | 是 | 测试库的id。 |
| plan_id | String | 是 | 测试计划的id。 |
| case_id | String | 是 | 测试用例的id。 |
| executor_id | String | 否 | 执行人的id。 |
