# 获取执行用例执行结果列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/testhub/run/statuses?library_id={library_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| library_id | String | 是 | 测试库的id。 |
