# 计划

## API 列表

- [创建一个计划](创建一个计划.md) - `POST /v1/testhub/libraries/{library_id}/plans`
- [部分更新一个计划](部分更新一个计划.md) - `PATCH /v1/testhub/libraries/{library_id}/plans/{plan_id}`
- [获取计划列表](获取计划列表.md) - `GET /v1/testhub/libraries/{library_id}/plans`
- [获取计划类型列表](获取计划类型列表.md) - `GET /v1/testhub/libraries/{library_id}/plan_types`
- [创建一个执行用例](创建一个执行用例.md) - `POST /v1/testhub/runs`
- [批量创建执行用例](批量创建执行用例.md) - `POST /v1/testhub/runs/bulk`
- [部分更新一个执行用例](部分更新一个执行用例.md) - `PATCH /v1/testhub/runs/{run_id}`
- [批量部分更新执行用例](批量部分更新执行用例.md) - `PATCH /v1/testhub/runs/bulk`
- [全量更新一个执行用例](全量更新一个执行用例.md) - `PUT /v1/testhub/runs/{run_id}`
- [批量操作执行用例](批量操作执行用例.md) - `POST /v1/testhub/libraries/{library_id}/plans/{plan_id}/runs/bulk`
- [获取执行用例列表](获取执行用例列表.md) - `GET /v1/testhub/runs`
- [获取执行用例执行结果列表](获取执行用例执行结果列表.md) - `GET /v1/testhub/run/statuses?library_id={library_id}`
- [获取执行用例的结果记录](获取执行用例的结果记录.md) - `GET /v1/testhub/runs/{run_id}/histories`
