# 获取计划类型列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/testhub/libraries/{library_id}/plan_types`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| library_id | String | 是 | 测试库的id。 |
