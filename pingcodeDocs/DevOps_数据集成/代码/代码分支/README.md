# 代码分支

## API 列表

- [创建一个代码分支](创建一个代码分支.md) - `POST /v1/scm/products/{product_id}/repositories/{repository_id}/branches`
- [部分更新一个代码分支](部分更新一个代码分支.md) - `PATCH /v1/scm/products/{product_id}/repositories/{repository_id}/branches/{branch_id}`
- [删除一个代码分支](删除一个代码分支.md) - `DELETE /v1/scm/products/{product_id}/repositories/{repository_id}/branches/{branch_id}`
- [获取代码分支列表](获取代码分支列表.md) - `GET /v1/scm/products/{product_id}/repositories/{repository_id}/branches`
