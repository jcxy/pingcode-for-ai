# 获取代码分支列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/scm/products/{product_id}/repositories/{repository_id}/branches`
- **版本**: 1.0.0

## 权限要求

- 企业令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| product_id | String | 是 | 代码托管平台的id。 |
| repository_id | String | 是 | 代码仓库的id。 |

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 否 | 分支的名称。 |
| work_item_id | String | 否 | 工作项的id。 |
