# 获取托管平台用户列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/scm/products/{product_id}/users`
- **版本**: 1.0.0

## 权限要求

- 企业令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| product_id | String | 是 | 代码托管平台的id。 |

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 否 | 代码托管平台上的用户名。 |
