# 获取托管平台列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/scm/products`
- **版本**: 1.0.0

## 权限要求

- 企业令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 否 | 代码托管平台的名称。 |

## 成功响应

### 响应示例：

```json
{
  "page_size": 30,
  "page_index": 0,
  "total": 1,
  "values": [
    {
      "id": "564587fe700d43b81b080765",
      "url": "https://rest_api_root/v1/scm/products/564587fe700d43b81b080765",
      "name": "Github",
      "type": "github",
      "description": "Github公有云"
    }
  ]
}
```
