# 获取提交列表

## 基本信息

- **方法**: `get`
- **路径**: `/v1/scm/commits`
- **版本**: 1.0.0

## 权限要求

- 企业令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| sha | String | 否 | 提交的SHA值。 |
| work_item_id | String | 否 | 工作项的id。 |
