# 代码仓库

## API 列表

- [创建一个代码仓库](创建一个代码仓库.md) - `POST /v1/scm/products/{product_id}/repositories`
- [全量更新一个代码仓库](全量更新一个代码仓库.md) - `PUT /v1/scm/products/{product_id}/repositories/{repository_id}`
- [部分更新一个代码仓库](部分更新一个代码仓库.md) - `PATCH /v1/scm/products/{product_id}/repositories/{repository_id}`
- [获取代码仓库列表](获取代码仓库列表.md) - `GET /v1/scm/products/{product_id}/repositories`
