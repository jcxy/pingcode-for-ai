# 构建记录

## API 列表

- [创建一条构建记录](创建一条构建记录.md) - `POST /v1/build/builds`
- [全量更新一条构建记录](全量更新一条构建记录.md) - `PUT /v1/build/builds/{build_id}`
- [部分更新一条构建记录](部分更新一条构建记录.md) - `PATCH /v1/build/builds/{build_id}`
- [删除一条构建记录](删除一条构建记录.md) - `DEL /v1/build/builds/{build_id}`
- [获取构建记录列表](获取构建记录列表.md) - `GET /v1/build/builds`
