# 删除一条构建记录

## 基本信息

- **方法**: `DEL`
- **路径**: `/v1/build/builds/{build_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| build_id | String | 是 | 构建的id。 |
