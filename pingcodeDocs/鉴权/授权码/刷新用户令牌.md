# 刷新用户令牌

## 基本信息

- **方法**: `get`
- **路径**: `/v1/auth/token?grant_type=refresh_token`
- **版本**: 1.0.0

## 描述

通过`refresh_token`重新获取`access_token`，可以避免用户频繁重新授权。`refresh_token`的有效期为90天，删除PingCode的应用或重置应用的Secret都会导致`refresh_token`失效。

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| grant_type | String | 是 | OAuth2的授予类型，这里固定为：`refresh_token`。 |
| refresh_token | String | 是 | 通过`authorization_code`获取`access_token`时，一起返回的`refresh_token`。 |

## 成功响应

### 响应示例：

```json
{
  "access_token": "e7321ca8-f724-4abd-9169-d76d095c6acf",
  "token_type": "Bearer",
  "expires_in": 1577808000
}
```
