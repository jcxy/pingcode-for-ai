# 项目

## API 列表

- [创建一个项目](创建一个项目.md) - `POST /v1/project/projects`
- [复制一个项目](复制一个项目.md) - `POST /v1/project/projects/{project_id}/clone`
- [部分更新一个项目](部分更新一个项目.md) - `PATCH /v1/project/projects/{project_id}`
- [获取项目列表](获取项目列表.md) - `GET /v1/project/projects`
- [获取项目状态列表](获取项目状态列表.md) - `GET /v1/project/project/states?project_id={project_id}`
- [向项目中添加一个成员](向项目中添加一个成员.md) - `POST /v1/project/projects/{project_id}/members`
- [部分更新一个项目内的成员](部分更新一个项目内的成员.md) - `PATCH /v1/project/projects/{project_id}/members/{member_id}`
- [获取项目中的成员列表](获取项目中的成员列表.md) - `GET /v1/project/projects/{project_id}/members`
- [在项目中移除一个成员](在项目中移除一个成员.md) - `DELETE /v1/project/projects/{project_id}/members/{member_id}`
- [向项目中添加一个项目属性](向项目中添加一个项目属性.md) - `POST /v1/project/projects/{project_id}/project_properties`
- [获取项目中的项目属性列表](获取项目中的项目属性列表.md) - `GET /v1/project/projects/{project_id}/project_properties`
- [在项目中移除一个项目属性](在项目中移除一个项目属性.md) - `DELETE /v1/project/projects/{project_id}/project_properties/{property_id}`
- [获取一个项目进度](获取一个项目进度.md) - `GET /v1/project/projects/{project_id}/progress`
