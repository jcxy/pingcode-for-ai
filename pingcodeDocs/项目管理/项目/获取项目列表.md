# 获取项目列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/project/projects`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| identifier | String | 否 | 项目的标识。 |
| type | String | 否 | 项目的类型。 |
| include_deleted | Boolean | 否 | 是否查询已删除的项目。该值默认为`false`。 |
| include_archived | Boolean | 否 | 是否查询已归档的项目。该值默认为`false`。 |
