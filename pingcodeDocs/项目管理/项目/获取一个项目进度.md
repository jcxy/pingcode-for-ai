# 获取一个项目进度

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/project/projects/{project_id}/progress`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 是 | 项目的id。 |

## 成功响应

### 响应示例：

```json
{
  "work_item": {
    "total": 4,
    "pending_count": 1,
    "in_progress_count": 2,
    "completed_count": 1
  }
}
```
