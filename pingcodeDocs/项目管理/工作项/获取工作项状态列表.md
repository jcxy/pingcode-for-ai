# 获取工作项状态列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/project/work_item/states?project_id={project_id}&work_item_type_id={work_item_type_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 是 | 项目的id。 |
| work_item_type_id | String | 是 | 工作项类型的id。工作项类型分为9种系统类型和一些自定义类型。系统类型包括：史诗、特性、用户故事、阶段、里程碑、需求、任务、缺陷和事务。可以通过`获取全部工作项类型列表`获得。 |
