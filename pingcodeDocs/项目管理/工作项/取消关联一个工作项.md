# 取消关联一个工作项

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/project/work_items/{work_item_id}/relations/{relation_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| work_item_id | String | 是 | 工作项的id。 |
| relation_id | String | 是 | 工作项关联的id。 |
