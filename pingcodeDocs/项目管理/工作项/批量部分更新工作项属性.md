# 批量部分更新工作项属性

## 基本信息

- **方法**: `PATCH`
- **路径**: `/v1/project/work_items`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| ids | String[] | 是 | 需要更新的工作项的id列表。最多可以批量更新100个工作项。 |
| property_name | String | 是 | 需要更新的工作项属性名。 |
| property_value | String | 否 | 需要更新的工作项属性值。 |

## 成功响应

### 响应示例：

```json
{
  "inserts": 0,
  "updates": 1,
  "deletes": 0
}
```
