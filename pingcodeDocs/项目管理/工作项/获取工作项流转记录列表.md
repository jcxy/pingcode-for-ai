# 获取工作项流转记录列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/project/work_items/{work_item_id}/transition_histories`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| work_item_id | String | 是 | 工作项的id。 |
