# 删除一个工作项工时

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/project/workloads/{workload_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| workload_id | String | 是 | 工时的id。 |
