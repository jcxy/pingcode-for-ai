# 获取工作项优先级列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/project/work_item/priorities?project_id={project_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 是 | 项目的id。 |

## 成功响应

```json
{
  "page_size": 30,
  "page_index": 0,
  "total": 1,
  "values": [

      "id": "5eb623f6a70571487ea47111",

```
