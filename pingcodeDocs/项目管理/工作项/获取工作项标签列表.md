# 获取工作项标签列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/project/work_item/tags`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 是 | 项目的id。 |
| name | String | 否 | 标签的名称。 |

## 成功响应

### 响应示例：

```json
{
  "page_size": 30,
  "page_index": 0,
  "total": 1,
  "values": [
    {
      "id": "5e6b35de50ef8153c2062f70",
      "url": "https://rest_api_root/v1/project/work_item_tags/5e6b35de50ef8153c2062f70",
      "name": "标签-1"
    }
  ]
}
```
