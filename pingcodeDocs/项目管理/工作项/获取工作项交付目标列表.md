# 获取工作项交付目标列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/project/deliverables`

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 否 | 项目的id。项目类型必须为`waterfall`或`hybrid`。 |
| work_item_id | String | 否 | 工作项的id。 |
