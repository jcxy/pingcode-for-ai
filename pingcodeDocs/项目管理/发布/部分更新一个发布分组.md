# 部分更新一个发布分组

## 基本信息

- **方法**: `PATCH`
- **路径**: `/v1/project/projects/{project_id}/version_sections/{section_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 是 | 项目的id。 |
| section_id | String | 是 | 发布分组的id。 |

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 是 | 发布分组的名称。 |
| description | String | 否 | 发布分组的描述。 |
