# 发布

## API 列表

- [创建一个发布](创建一个发布.md) - `POST /v1/project/projects/{project_id}/versions`

- [部分更新一个发布](部分更新一个发布.md) - `PATCH /v1/project/projects/{project_id}/versions/{version_id}`
- [获取发布列表](获取发布列表.md) - `GET /v1/project/projects/{project_id}/versions`
- [创建一个发布分组](创建一个发布分组.md) - `POST /v1/project/projects/{project_id}/version_sections`
- [部分更新一个发布分组](部分更新一个发布分组.md) - `PATCH /v1/project/projects/{project_id}/version_sections/{section_id}`
- [获取发布分组列表](获取发布分组列表.md) - `GET /v1/project/projects/{project_id}/version_sections`
- [删除一个发布分组](删除一个发布分组.md) - `DELETE /v1/project/projects/{project_id}/version_sections/{section_id}`
- [创建一个发布类别](创建一个发布类别.md) - `POST /v1/project/projects/{project_id}/version_categories`
- [部分更新一个发布类别](部分更新一个发布类别.md) - `PATCH /v1/project/projects/{project_id}/version_categories/{version_category_id}`
- [获取发布类别列表](获取发布类别列表.md) - `GET /v1/project/projects/{project_id}/version_categories`
- [删除一个发布类别](删除一个发布类别.md) - `DELETE /v1/project/projects/{project_id}/version_categories/{version_category_id}`
- [创建一个发布阶段](创建一个发布阶段.md) - `POST /v1/project/stages`
- [部分更新一个发布阶段](部分更新一个发布阶段.md) - `PATCH /v1/project/stages/{stage_id}`
- [获取发布阶段列表](获取发布阶段列表.md) - `GET /v1/project/stages`
- [删除一个发布阶段](删除一个发布阶段.md) - `DELETE /v1/project/stages/{stage_id}`
