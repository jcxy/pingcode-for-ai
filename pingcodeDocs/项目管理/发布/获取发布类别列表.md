# 获取发布类别列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/project/projects/{project_id}/version_categories`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 是 | 项目的id。 |
