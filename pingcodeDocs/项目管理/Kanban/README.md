# Kanban

## API 列表

- [创建一个看板](创建一个看板.md) - `POST /v1/project/projects/{project_id}/boards`
- [部分更新一个看板](部分更新一个看板.md) - `PATCH /v1/project/projects/{project_id}/boards/{board_id}`
- [获取看板列表](获取看板列表.md) - `GET /v1/project/projects/{project_id}/boards`
- [删除一个看板](删除一个看板.md) - `DELETE /v1/project/projects/{project_id}/boards/{board_id}`
- [创建一个看板栏](创建一个看板栏.md) - `POST /v1/project/projects/{project_id}/boards/{board_id}/entries`
- [部分更新一个看板栏](部分更新一个看板栏.md) - `PATCH /v1/project/projects/{project_id}/boards/{board_id}/entries/{entry_id}`
- [获取看板栏列表](获取看板栏列表.md) - `GET /v1/project/projects/{project_id}/boards/{board_id}/entries`
- [删除一个看板栏](删除一个看板栏.md) - `DELETE /v1/project/projects/{project_id}/boards/{board_id}/entries/{entry_id}`
- [创建一个泳道](创建一个泳道.md) - `POST /v1/project/projects/{project_id}/boards/{board_id}/swimlanes`
- [部分更新一个泳道](部分更新一个泳道.md) - `PATCH /v1/project/projects/{project_id}/boards/{board_id}/swimlanes/{swimlane_id}`
- [获取泳道列表](获取泳道列表.md) - `GET /v1/project/projects/{project_id}/boards/{board_id}/swimlanes`
- [删除一个泳道](删除一个泳道.md) - `DELETE /v1/project/projects/{project_id}/boards/{board_id}/swimlanes/{swimlane_id}`
