# 删除一个看板栏

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/project/projects/{project_id}/boards/{board_id}/entries/{entry_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 是 | 项目的id。 |
| board_id | String | 是 | 看板的id。 |
| entry_id | String | 是 | 看板栏的id。 |
