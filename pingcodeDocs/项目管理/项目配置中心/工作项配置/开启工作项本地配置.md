# 开启工作项本地配置

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/project/work_item_plans`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 是 | 项目的id。 |
