# 获取状态方案中的工作项状态列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/project/work_item_state_plans/{state_plan_id}/work_item_states`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| state_plan_id | String | 是 | 工作项状态方案的id。 |
