# 向工作项类型方案中添加一个工作项类型

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/project/work_item_type_plans/{work_item_type_plan_id}/work_item_types`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| work_item_type_plan_id | String | 是 | 工作项类型方案的id。 |

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| work_item_type_id | String | 是 | 工作项类型的id。 |
