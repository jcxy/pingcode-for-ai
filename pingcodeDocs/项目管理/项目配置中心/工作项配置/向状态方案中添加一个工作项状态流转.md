# 向状态方案中添加一个工作项状态流转

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/project/work_item_state_plans/{state_plan_id}/work_item_state_flows`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| state_plan_id | String | 是 | 工作项状态方案的id。 |

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| from_state_id | String | 是 | 起始工作项状态的id。 |
| to_state_id | String | 是 | 可更改为的工作项状态的id。 |
