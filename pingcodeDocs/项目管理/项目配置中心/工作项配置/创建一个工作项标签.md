# 创建一个工作项标签

## 基本信息

- **方法**: `POST`
- **路径**: `/v1/project/work_item_tags`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 是 | 标签的名称。在一个企业中这个名称是唯一的。 |

## 成功响应

### 响应示例：

```json
{
  "id": "5e6b35de50ef8153c2062f70",
  "url": "https://rest_api_root/v1/project/work_item_tags/5e6b35de50ef8153c2062f70",
  "name": "标签-1",
  "color": "#56ABFB"
}
```
