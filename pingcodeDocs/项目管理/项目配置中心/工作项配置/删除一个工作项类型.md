# 删除一个工作项类型

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/project/work_item_types/{work_item_type_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### Parameter

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| work_item_type_id | String | 是 | 工作项类型的id。 |

## 成功响应

### 响应示例：

```json
{
  "id": "630da48bc9443b1aa94ce3df",
  "url": "https://rest_api_root/v1/project/work_item_types/630da48bc9443b1aa94ce3df",
  "name": "非功能性需求",
  "group": "requirement",
  "is_system": false
}
```
