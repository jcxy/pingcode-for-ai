# 获取迭代列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/project/projects/{project_id}/sprints`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| project_id | String | 是 | 项目的id。 |

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| name | String | 否 | 迭代的名称。 |
| status | String | 否 | 迭代的状态。 |
| created_between | String | 否 | 创建时间介于的时间范围，通过','分割起始时间。 |
| updated_between | String | 否 | 更新时间介于的时间范围，通过','分割起始时间。 |
