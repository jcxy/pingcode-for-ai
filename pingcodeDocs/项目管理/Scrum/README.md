# Scrum

## API 列表

- [创建一个迭代](创建一个迭代.md) - `POST /v1/project/projects/{project_id}/sprints`
- [批量创建迭代](批量创建迭代.md) - `POST /v1/project/sprints/bulk`
- [部分更新一个迭代](部分更新一个迭代.md) - `PATCH /v1/project/projects/{project_id}/sprints/{sprint_id}`
- [获取迭代列表](获取迭代列表.md) - `GET /v1/project/projects/{project_id}/sprints`
- [创建一个迭代分组](创建一个迭代分组.md) - `POST /v1/project/projects/{project_id}/sprint_sections`
- [部分更新一个迭代分组](部分更新一个迭代分组.md) - `PATCH /v1/project/projects/{project_id}/sprint_sections/{section_id}`
- [获取迭代分组列表](获取迭代分组列表.md) - `GET /v1/project/projects/{project_id}/sprint_sections`
- [删除一个迭代分组](删除一个迭代分组.md) - `DELETE /v1/project/projects/{project_id}/sprint_sections/{section_id}`
- [创建一个迭代类别](创建一个迭代类别.md) - `POST /v1/project/projects/{project_id}/sprint_categories`
- [部分更新一个迭代类别](部分更新一个迭代类别.md) - `PATCH /v1/project/projects/{project_id}/sprint_categories/{sprint_category_id}`
- [获取迭代类别列表](获取迭代类别列表.md) - `GET /v1/project/projects/{project_id}/sprint_categories`
- [删除一个迭代类别](删除一个迭代类别.md) - `DELETE /v1/project/projects/{project_id}/sprint_categories/{sprint_category_id}`
