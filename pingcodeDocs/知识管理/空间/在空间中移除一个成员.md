# 在空间中移除一个成员

## 基本信息

- **方法**: `DELETE`
- **路径**: `/v1/wiki/spaces/{space_id}/members/{member_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| space_id | String | 是 | 空间的id。 |
| member_id | String | 是 | 企业成员或团队的id。 |
