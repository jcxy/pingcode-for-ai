# 空间

## API 列表

- [创建一个空间](创建一个空间.md) - `POST /v1/wiki/spaces`
- [部分更新一个空间](部分更新一个空间.md) - `PATCH /v1/wiki/spaces/{space_id}`
- [删除一个空间](删除一个空间.md) - `DELETE /v1/wiki/spaces/{space_id}`
- [获取空间列表](获取空间列表.md) - `GET /v1/wiki/spaces`
- [向空间中添加一个成员](向空间中添加一个成员.md) - `POST /v1/wiki/spaces/{space_id}/members`
- [在空间中移除一个成员](在空间中移除一个成员.md) - `DELETE /v1/wiki/spaces/{space_id}/members/{member_id}`
- [获取空间中的成员列表](获取空间中的成员列表.md) - `GET /v1/wiki/spaces/{space_id}/members`
