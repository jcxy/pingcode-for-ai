# 获取页面列表

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/wiki/pages?space_id={space_id}`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| space_id | String | 是 | 空间的id。 |
| parent_id | String | 否 | 父页面的id。 |
| created_between | String | 否 | 创建时间介于的时间范围，通过','分割起始时间。 |
| updated_between | String | 否 | 更新时间介于的时间范围，通过','分割起始时间。 |
