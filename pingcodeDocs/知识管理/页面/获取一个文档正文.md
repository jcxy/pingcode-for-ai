# 获取一个文档正文

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/wiki/pages/{page_id}/content`
- **版本**: 1.0.0

## 权限要求

- 企业令牌/用户令牌

## 参数

### 路径参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| page_id | String | 是 | 页面的id。 |

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| format_type | String | 否 | 正文格式。 |
| version_id | String | 否 | 页面版本的id。默认值为页面当前版本的id。 |
