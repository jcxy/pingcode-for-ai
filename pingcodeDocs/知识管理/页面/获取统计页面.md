# 获取统计页面

## 基本信息

- **方法**: `GET`
- **路径**: `/v1/dashboard/pages`
- **版本**: 1.0.0

## 权限要求

- 用户令牌

## 参数

### 查询参数

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| category | String | 是 | 统计页面的类型。 |
| count | String | 否 | 统计页面的数量。 |
