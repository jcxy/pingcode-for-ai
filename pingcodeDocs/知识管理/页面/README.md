# 页面

## API 列表

- [创建一个页面](创建一个页面.md) - `POST /v1/wiki/pages`
- [部分更新一个页面](部分更新一个页面.md) - `PATCH /v1/wiki/pages/{page_id}`
- [删除一个页面](删除一个页面.md) - `DELETE /v1/wiki/pages/{page_id}`
- [获取页面列表](获取页面列表.md) - `GET /v1/wiki/pages?space_id={space_id}`
- [获取一个文档正文](获取一个文档正文.md) - `GET /v1/wiki/pages/{page_id}/content`
- [更新一个文档正文](更新一个文档正文.md) - `PUT /v1/wiki/pages/{page_id}/content`
- [获取一个页面的版本列表](获取一个页面的版本列表.md) - `GET /v1/wiki/pages/{page_id}/versions`
- [恢复一个页面到指定版本](恢复一个页面到指定版本.md) - `POST /v1/wiki/pages/{page_id}/versions/{version_id}/restore`
- [获取统计页面](获取统计页面.md) - `GET /v1/dashboard/pages`
